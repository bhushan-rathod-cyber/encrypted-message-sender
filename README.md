# 🔐 Encrypted–Decrypted Message Sender

A secure communication system that encrypts and decrypts messages using basic cryptographic techniques.

## 🚀 Features
- Encrypt messages
- Decrypt messages
- Simple web interface

## 🛠 Technologies Used
- Python
- Flask
- HTML/CSS

## ▶️ How to Run
1. Install Python
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open browser at localhost

## 📌 Future Scope
- AES encryption
- RSA encryption
- End-to-End encryption
